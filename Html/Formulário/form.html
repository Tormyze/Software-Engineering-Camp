Formulário?

Bom, claro né. Vamo falar de uma das estruturas funcionais que o Html nos permite criar: o formulário.

Para criar um formulário (semântico form) devemos fazer a seguinte estrutura básica:


(form action="" method="")


(/form)


* action="" descreve para onde os dados serão enviados.

* method="" descreve se os dados serão codificados e como serão para serem enviados.


Ele nos permite criar diversos tipos de entradas onde os usuários podem nos enviar dados ou à algum local.
