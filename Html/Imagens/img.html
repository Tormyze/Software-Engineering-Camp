How to add images??

Sim, isso vai ser em inglês KSKKS

First, you'll need to add your image as a file here in the program or get one from the internet. Then, you'll use the img tag!

You can attach an image to the tag by two ways:

  • through the ABSOLUTE way.
  • through the RELATIVE way.

ABSOLUTE means the way where you can put the entire address of the image's webpage in the document.

RELATIVE means that you just need to locate the position of the file somewhere in the program and put that in the tag.

Use the syntax: (img src="" alt="")

  • src="", put the image's file inside here.
  • alt="", put any random text here in case the image doesn't appear.

And that's it.
