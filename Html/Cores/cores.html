Cores?

Pois é, meu amigão, você também consegue usar umas cores pelo código, da maneira que quiser. Dá uma olhada:


Atributos de cor

Você pode adicionar um atributo de cor com seu devido valor para alterar a coloração de uma propriedade ou item.

Exemplo:

  • Utilizando o atributo bgcolor=" " em body, pra mudar o background.

Atributo style=" "

É possível simular o CSS em um item através de um atributo chamado STYLE. Coloque ele dentro do elemento e, em seguida, adicione uma propriedade nas aspas " " com algum valor de cor.

Exemplo:

  • Cor vermelha

Sistema RGB

No caso, RGB é o sistema de cores com RED, GREEN e BLUE. Cada uma possui nível de intensidade do 0 ao 255. Misturando, chegamos em várias cores diferentes.

Exemplo:

  • Vermelho
  • Verde
  • Azul

  • Amarelo
  • Magenta
  • Ciano

  • Preto
  • Cinza
  • Branco

Sistema HEX

Há também o sistema hexadecimal de cores, chamado de HEX. Ele utiliza os caracteres de 0 - 9 e A - F como pares de valores de intensidade de cada cor. Bora explicar melhor:

#

RR

GG

BB

Cada par representa uma cor do RGB.

A menor intensidade é 0 e a maior é F. Assim, podemos, por exemplo, criar as seguintes cores:


Amarelo HEX

Para o amarelo, combine vermelho puro com verde puro, usando #FFFF00.

Magenta HEX

Para o magenta, combine vermelho puro com azul puro, usando #FF00FF.

Ciano HEX

Para o ciano, combine verde puro com azul puro, usando #00FFFF.

Cinza HEX

Para o cinza, coloque a metade da intensidade em cada um dos pares, que no caso é #888888.

Com o sistema hexadecimal é possível misturar variadas cores com forças diferentes, obtendo diversos resultados interessantes.

Sistema HSL

Não menos importante, temos também um sistema chamado HSL (hue, saturation, lightness). Com ele, é possível definir a SATURAÇÃO e a LUMINOSIDADE da cor desejada.

  • Saturação
    • Intensidade total da cor.
        • Vai de cinza até a cor total.

  • Luminosidade
    • Brilho da cor.
        • Vai do preto até o puro branco.

Funciona da seguinte forma:

O valor de hue é um ângulo em uma roda de cores planificada.

  • 0° é vermelho.
  • 120° é verde.
  • 240° é azul.

Para os valores de saturação e luminosidade respectivamente, é utilizado a porcentagem (%).

Assim, podemos estruturar um código de HSL dessa forma:

    hsl(0, 100%, 50%)
    Essa é a sintaxe!
    hsl(30, 75%, 30%)
    Seja criativo, teste os valores.
    hsl(120, 100%, 25%)
    Tons escuros e claros.
