Css


∆ Texto

  • text-align: ;
    • alinhamento horizontal do texto
      • center
      • left
      • right
      • justify

  • vertical-align: ;
    • alinhamento vertical do texto
      • baseline
      • text-top
      • text-bottom
      • sub
      • super


    • text-decoration: ;
      • shorthand para:

        text-decoration-line: ;

        • adiciona uma linha no próprio texto
          • overline
          • line-through
          • underline
          • overline underline

        text-decoration-color: ;

        • muda a cor da linha do próprio texto
          • cor predefinida
          • RGB HSL HEX


  • text-transform: ;
    • maiúsculo, minúsculo e capitalizado
      • uppercase
      • lowercase
      • capitalize

  • text-indent: ;
    • aumenta o tamanho da indentação
      • px

  • letter-spacing: ;
    • espaçamento entre letras
      • Z (números inteiros)
      • 1.0, 1.8, 0.7, etc

  • word-spacing: ;
    • espaço entre palavras
      • px

  • line-height: ;
    • espaçamento entre linhas
      • px

  • white-space: ;
    • define se o texto vai continuar sempre na mesma linha ou vai quebrar em várias cabíveis
      • "nowrap;" para nunca quebrar linhas

  • text-shadow: ;
    • aplica sombra ao texto
    • px, valores de cores
      • 1° px -> sombra horizontal
      • 2° px -> sombra vertical
      • 3° px -> efeito de blur
      • cor -> cor da sombra
        • "1px 2px 5px red;"

∆ Links

  1. a:link {}
    2. seletor do link inerte

  2. a:visited {}
    3. seletor do link que já foi visitado alguma vez

  3. a:hover {}
    4. seletor do link enquanto o mouse está sobre ele

  4. a:active {}
    5. seletor do link quando é clicado

  • color: ;
    • muda a cor do link

    Quaisquer outras propriedades decorativas, altere entre os tipos de seletores

    • backgrounds e cores
    • comportamentos de posição
    • sombra e efeitos
    • etc

∆ Localizadores

  • class=" "
    • atributo que infere um valor universal que pode ser usado em qualquer e mais de um elemento como parâmetro para ser/serem modificado(s) no Css

  • id=" "
    • atributo que define um valor exclusivo a um elemento como um parâmetro que pode ser usado para modificações no Css

∆ Posição

  • position: ;
    • tag com 1 valor definido, altera a posição do elemento

      • static; (padrão)
        • o elemento possui sua posição natural padrão

      • relative;
        • o elemento se posiciona por valores em relação a ele mesmo

      • absolute;
        • um elemento com absolute, quando está dentro de outro elemento, sempre seguirá a posição deste elemento

      • fixed;
        • o elemento está absolutamente fixo à sua posição na página e nunca mudará dela

      • sticky;
        • o elemento mantém sua posição até o momento em que estiver prestes à sair da tela, mas fica grudado nela, em sua borda


  • top: ;
    • muda a posição superior
      • px

  • left: ;
    • muda a posição esquerda
      • px

  • bottom: ;
    • muda a posição inferior
      • px

  • right: ;
    • muda a posição direita
      • px

∆ Overflow/"Estouro" do conteúdo

  • overflow: ;
    • controla se o conteúdo vai respeitar o espaço ou estourá-lo

    • visible (padrão)
      • o conteúdo continuará seu seguimento mesmo após estourar o espaço da página ou do elemento em que está

    • hidden
      • o conteúdo literalmente sofrerá um corte na sua extremidade estourada para caber no espaço

    • scroll
      • serão obrigatoriamente adicionados 2 scrolls móveis para o elemento estourado

    • auto
      • add apenas o(s) scroll(s) necessário(s) ao elemento estourado

∆ Float & Clear

  • float: ;
    • usado para colocar elementos ao lado de outros e puxá-los, normalmente para colocar uma imagem a um lado qualquer de um texto
      • left
      • right
      • none
      • inherit

  • clear: ;
    • controla o que acontece com os elementos ao lado dos com float
      • none (padrão)
        • segue o elemento float

      • left
        • se posiciona embaixo-esquerda do elemento float

      • right
        • se posiciona embaixo-direita do elemento float

      • both
        • se posiciona embaixo-esquerda/direita do elemento float, se centralizando aos dois

      • inherit
        • se posiciona seguindo o elemento pai
