Cores e Backgrounds

Agora com esse lindo Css em mãos a gente pode formatar nosso documento do jeito que quisermos.

Sendo assim, bora começar com artes visuais ao nosso doc, haha.

Aplicando cores em tudo!

Pra começar nossa brincadeira artística, vamo falar de umas propriedades-chave pra nós usarmos no Css:


  • color: ;
    • cor do texto


  • background: ;
    • atalho para todas as aplicações no background


  • background-color: ;
    • cor do fundo


  • background-image: ;
    • add uma imagem como fundo
    • atributo url( ) para add a imagem!


  • background-size: ;
    • tamanho da(s) imagem(ns)
    • vírgula para separar atribuições à cada imagem
    • X e Y, horizontal e vertical
      • auto cover contain
      • units {px, %, cm, ...}


  • background-attachment: ;
    • define se o fundo irá se mover junto com o scroll do mouse
      • scroll - vai se mover
      • fixed - não vai se mover
      • local - vai se mover junto com os conteúdos do mesmo elemento


  • background-position: ;
    • posição do fundo
    • X e Y, horizontal e vertical
      • left right center etc
      • units {px, %, cm, ...}


  • background-repeat: ;
    • se a img vai repetir pelo fundo
      • repeat
      • repeat-x
      • repeat-y
      • no-repeat
      • space
      • round

Mas é claro que não existem só estes, muito pelo contrário.

Que comece a busca pelas properties do Css!
